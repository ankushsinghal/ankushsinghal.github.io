Bubble Sort


Bubble Sort is a sorting algorithm that sorts the elements of an array in order, by repeatedly swapping adjacent elements in the array.

The algorithm repeatedly steps through the array to be sorted, compares each pair of adjacent items and swaps them if they are in wrong order. The pass through the array is repeated until no swaps are needed, which indiactes that the array is sorted.


Some of the features of Bubble sort algorithm are as follows:

  • Bubble sort is a stable algorithm, which means that it does not change the relative ordering of the elements with equal keys.
  • Bubble sort can do sorting in-place and thus does not require any additional memory space.


The psuedocode for the Bubble sort algoritm is as follows:


BUBBLE-SORT(A) :
    for i = 0 to (A.length - 2) :
        for j = 0 to (A.length - i - 2) :
            if A[j] > A[j + 1] :
                swap(A[j], A[j + 1])

  • Worst Case Performance : Ο(n2)
  • Best Case Performance : Ο(n)
  • Average Case Performance : Ο(n2)
  • Worst Case Space Complexity : Ο(n) total, Ο(1) auxillary
