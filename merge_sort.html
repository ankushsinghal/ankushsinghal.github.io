Merge Sort


Merge sort is a divide and conquer algorithm which sorts a given array.

Merge sort divides the array into two halves, sorts the two halves recursively using merge sort and then merges the two halvs to form the final sorted array.


Some of the features of Merge sort algorithm are as follows :

  • Merge sort is a stable algorithm.
  • Merge sort is not in-place, and thus requires additional memory space to sort the array.

The psuedocode for Merge sort is as follows : 


MERGE-SORT(A,p,r) : 
    if p < r :
        q = floor((p + r) / 2)
        MERGE-SORT(A,p,q)
        MERGE-SORT(A,q+1,r)
        MERGE(A,p,q,r)

The above "MERGE-SORT" function uses a "MERGE" function to merge the arrays. The psuedocode for "MERGE" is as follows : 


MERGE(A,p,q,r) : 
    n1 = q - p + 1
    n2 = r - q
    let L[0...n1] of length (n1 + 1) and R[0...n2] of length (n2 + 1) be two new arrays
    for i = 0 to (n1 - 1) :
        L[i] = A[p + i - 1]
    for j = 0 to (n2 - 1) :
        R[j] = A[q + j]
    L[n1] = ∞             //Sentinal node
    R[n2] = ∞             //Sentinal node
    i = 0
    j = 0
    for k = p to r :
        if L[i] <= R[j] :
            A[k] = L[i]
            i = i + 1
        else :
            A[k] = R[j]
            j = j+1

  • Worst Case Performance : Ο(n log n)
  • Best Case Performance : Ο(n log n)
  • Average Case Performance : Ο(n log n)
  • Worst Case Space Complexity : Ο(n) total, Ο(n) auxillary
