Selection Sort


Selection Sort is a sorting algorithm that sorts the elements of an array in order, by repeatedly finding the minimum element of the unsorted array.

The algorithm divides the array into two parts, the sorted array which is built from left to right and the remaining unsorted array. The sorted array is initially empty. In each iteration, the minimum element is found from the unsorted array and is put at the end of the sorted array.


Some of the features of Insertion sort algorithm are as follows:

  • Selection sort is a stable algorithm, which means that it does not change the relative ordering of the elements with equal keys.
  • Selection sort can do sorting in-place and thus does not require any additional memory space.
  • Selection sort never makes more than Ο(n) swaps.


The psuedocode for the Selection sort algoritm is as follows:


SELECTION-SORT(A) :
    for j = 0 to (A.length - 2) :
        index_min = j
        for i = j + 1 to n - 1 :
            if A[i] < A[index_min] :
                index_min = i
        if index_min != j :
            swap(A[j], A[index_min])

  • Worst Case Performance : Ο(n2)
  • Best Case Performance : Ο(n2)
  • Average Case Performance : Ο(n2)
  • Worst Case Space Complexity : Ο(n) total, Ο(1) auxillary
