Insertion Sort


Insertion Sort is a sorting algorithm that sorts the elements of an array in order one at a time.

Insertion sort works just like many people sort a hand of playing cards. Initially our left hand is empty, and all our cards are down on the table. We keep on picking a card at a time from the table and put it in the appropriate position in our left hand, so that all the cards in our left hand are sorted.


Some of the features of Insertion sort algorithm are as follows:

  • Insertion sort is more efficient in practice than most other quadratic algorithms, such as selection sort and bubble sort.
  • Insertion sort is usually faster in pratice than other asymptotically faster algorithms for small data sets, but is much less efficient on large lists.
  • Insertion sort is a stable algorithm, which means that it does not change the relative ordering of the elements with equal keys.
  • Insertion sort can do sorting in-place and thus does not require any additional memory space.

The psuedocode for the Insertion sort algoritm is as follows:


INSERTION-SORT(A) :
    for j = 1 to (A.length - 1) :
        key = A[j]
        //Insert A[j] into sorted sequence A[0...j-1]
        i = j - 1
        while i >= 0 and A[i] > key :
            A[i + 1] = A[i]
            i = i - 1
        A[i + 1] = key

  • Worst Case Performance : Ο(n2) comparisons, swaps
  • Best Case Performance : Ω(n) comparisons, Ο(1) swaps
  • Average Case Performance : Ο(n2) comparisons, swaps
  • Worst Case Space Complexity : Ο(n) total, Ο(1) auxillary
